***********************************************************
Extreme XML Column Sample Code 05/2001
***********************************************************

Step 1: Install Visual Studio .NET Beta1 from http://msdn.microsoft.com/vstudio/nextgen/beta.asp

Step 2: Install MSXML 4.0 from http://msdn.microsoft.com/xml

Step 3: Regsvr32 (register) the POValidator.dll located in the PurchaseOrder directory

Step 4: Open the POTestHarness.sln solution file from the Bin directory

Step 5: Press F5 to start the project

The schema file is located in the PurchaseOrder folder.  The XML PO document is located in the Bin\POTest\bin\Debug folder.
