1.  To run the demos you will need to open an account with the service bus. 

2.  The demos all use “MySolution” for solution name. You will need to replace that with your solution name.

3.  Most of the demos default for CardsSpace credentials for authentication against the service bus. You can of course rework them for password or certificate, either in config or using the features of ServiceModelEx.

4. The security features of ServiceModelEx are discussed in future columns.
