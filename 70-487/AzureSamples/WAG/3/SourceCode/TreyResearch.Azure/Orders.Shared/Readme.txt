Orders.Shared

What does this project do?
----------------------------------------
The Orders.Shared project contains a library of reusable classes that wrap the existing MessageSender, MessageReceiver, and BrokeredMessages classes (amongst others) found in the Microsoft.ServiceBus.Messaging namespace. The purpose of these classes is to abstract the send and receive functionality so that all send and receive operations are performed asynchronously.
