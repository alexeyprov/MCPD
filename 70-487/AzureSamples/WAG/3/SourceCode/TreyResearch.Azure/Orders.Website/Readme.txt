Orders.Website

What does this project do?
---------------------------------------
This is a Windows Azure Web Role (ASP.NET MVC 3 project) that serves as the front end for TreyResearch.  Customers will be able to register, place and monitor their orders.


Project Dependencies
---------------------------------------
None.


What Projects depend on this project?
---------------------------------------
None.


How to use this project
---------------------------------------
When you execute the Orders.Azure project, the web role will launch automatically.  Browse through the product catalog and place new orders.
