Examples Summary

This document describes the examples that illustrate the scenarios described in the book Moving Applications to the Cloud on Windows Azure (3rd edition) (see http://wag.codeplex.com/). Each example is a Microsoft Visual Studio 2010 project that you can compile and run. For information about running each example, and the known issues and options available, see the Release Notes.

 

 

Before you start

 

Ensure you have a compatible version of SQL Server installed on your computer:

  • For Windows 8 and Windows Server 2012: SQL Server 2012 or SQL Server Express 2012
  • For Windows 7 and Windows Server 2008: SQL Server 2012, SQL Server Express 2012, SQL Server 2008, or SQL Server Express 2008

If you are not using SQL Server Express, you must edit the configuration file used by the Dependency Checker utility. See the Release Notes for details.

 

Run the CheckDependencies.cmd batch file that is located in the root folder of the examples to check for and install the software prerequisites for the examples.

 

For more information about the prerequisites, setup, and running the examples see the Release Notes.

 

 

About The Examples

 

The SourceCode folder of the examples contains the following folders and solutions:

 

BeforeAzure

This is the original on-premises solution before being adapted to run in Windows Azure.

 

Azure-CloudService-SQLServer

This is on-premises solution after it has been adapted to run in Windows Azure using SQL Server hosted in a Virtual Machine. Note that the example is initially configured to use a local SQL Server Express database by default.

 

Azure-CloudService-WASDB

This is the solution that uses Windows Azure SQL Database instead of SQL Server. See the Known Issues section of the Release Notes for information on creating a Windows Azure SQL Database instance.

 

Azure-WorkerRole

This is the solution that uses a Windows Azure Worker Role to perform background tasks.

 

Azure-TableStorage

This is the solution that uses Windows Azure Table storage instead of Windows Azure SQL Database. See the Known Issues section of the Release Notes for information about an error that may occur the first time you run this example in the compute emulator.

 

aExpense.AcceptanceTests

This folder contains a set of acceptance tests that you can run to verify the test cases for the examples. See the Release Notes for information on using these tests.

 

Shared

This folder contains shared code used by more than one of the examples. It includes the simulated token issuer that the examples use for authentication.
